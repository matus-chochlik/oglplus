(version 3 and higher)

Visit the OGLplus pages on , or for recent news and more info.

Introduction

OGLplus is an open-source, cross-platform library which implements an object-oriented facade over the OpenGL® (version 3 and higher) C-language API. It provides wrappers which automate resource and object management and make the use of OpenGL in C++ safer and more convenient. Read more...
